# 📰 RSS Aggregator

RSS Aggregator is a web application for reading RSS feeds.  
It allows users to add RSS sources, browse posts, and read articles in a convenient format.

This project was built as part of a learning process and demonstrates working with asynchronous requests, data parsing, and state management.

---

## 🚀 Demo

👉 https://frontend-project-11-neon-five.vercel.app/

---

## 🛠️ Tech Stack

- JavaScript (ES Modules)
- Vite
- Axios
- Yup (validation)
- i18next (localization)
- Bootstrap (UI)
- Valtio (state management)

---

## 📦 Installation and Setup

### 1. Clone the repository

```bash
git clone https://github.com/ilia-m-dev/frontend-project-11.git
cd frontend-project-11
```
### 2. Install dependencies

```bash
npm install
```

### 3. Run in development mode

```bash
npm run dev
```
